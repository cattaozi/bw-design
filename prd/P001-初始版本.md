## 背景

团队或家庭成员经常忘记彼此的生日，希望有一个简单的工具，打开即能看到近期谁要过生日，不需要任何操作。

## 目标用户

需要管理多人生日信息的个人用户（家庭、小团队），本地部署，单用户使用。

## 功能描述

**生日墙主页面**
- 默认展示未来 30 天内的生日，按日期升序排列
- 每个卡片显示：头像（未上传则显示姓名首字母）、姓名、生日日期（MM/DD）、距今 N 天
- 今日生日者在顶部单独区域高亮展示，配撒花动画
- 支持切换查看范围：未来 7 天 / 30 天（默认）/ 90 天 / 全部
- 卡片可选显示年龄

**人员管理**
- 支持增删改查生日信息
- 必填：姓名、生日（月日）；选填：年份、头像、备注
- 头像支持上传（jpg/png，最大 2MB，自动裁剪为圆形）
- 每位人员可独立配置是否显示年龄

**分组**
- 支持创建分组（如"家人"、"同事"、"朋友"），可设置颜色标签
- 主页面支持按分组筛选

**提醒**
- 浏览器 Tab 显示今日生日人数角标
- 支持邮件提醒，可配置提前 1 / 3 / 7 天发送
- 每天早上 8:00 自动检查并发送

**设置**
- 自定义应用名称、默认展示范围、日期格式
- 邮件提醒配置
- CSV 导入 / 导出

## 非功能要求

- 首屏加载 < 1.5s（100 人以内数据）
- 支持 Chrome / Safari / Firefox 最新两个大版本
- 支持移动端响应式布局
- 数据本地存储，不上传云端

## 不做什么

- 不做多用户 / 权限系统
- 不做云同步
- v1 不做微信 / 短信提醒（只做邮件）
- 不做生日祝福语自动发送
