## 用户故事

作为一个容易忘记日期的个人用户，我希望在有人今天或即将过生日时能收到明显提醒，这样即使没主动打开生日墙页面也不会错过为 TA 庆祝。

## 验收标准

- [ ] 当今日有生日人员时，浏览器 Tab 上显示今日生日人数的角标（如红点或数字）
- [ ] 当今日没有生日人员时，浏览器 Tab 恢复为默认标题样式（无角标或为 0）
- [ ] 支持配置邮件提醒提前天数：1 / 3 / 7 天，可多选或单选，未配置时不发送提醒
- [ ] 支持在设置中配置邮件发送相关信息（如收件邮箱等），未完成配置时在提醒设置区域给出明显提示
- [ ] 系统每天早上本地时间 8:00 自动检查未来 N 天（根据配置）内的生日并发送提醒邮件
- [ ] 每封提醒邮件中至少包含：姓名、具体生日日期、距今天数、所属分组（如有）
- [ ] 同一天内同一生日不会重复发送多封提醒邮件
- [ ] 邮件发送失败时有可观测的错误信息记录（如日志或简单提示），方便排查问题

## 备注

- v1 仅支持邮件提醒，不做微信/短信等其他渠道
- 如本地环境无法保证严格的 8:00 触发（例如应用未启动），需要约定并在文档中说明行为（如下次启动时补发或忽略）
